# TicTacToe-CPP
TicTacToe game done with C++ 

# How to execute
In order to compile and execute the files, what you have to do is:
1 - gcc -c auxiliarFunctions.cpp
2 - gcc -c main.cpp
3 - gcc -o <NameFile> auxiliarFunctions.o main.o
