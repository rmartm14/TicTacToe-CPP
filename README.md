# TicTacToe-CPP
TicTacToe game done with C++ 

# How to execute
In order to compile and execute the files, what you have to do is:
```bash
gcc -c auxiliarFunctions.cpp 
gcc -c main.cpp 
gcc -o <NameFile> auxiliarFunctions.o main.o 
```
